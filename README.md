# Spring Boot 온라인 시험 프로젝트

### - 주요 개발 환경
- `개발 도구 : eclipse sts 4.17`
- `자바 버전 : 11`
- `데이터 베이스 : MariaDB`
- `빌드 플러그인 : maven`
- `MyBatis`
- `Lombok`

### - 주요 기능
#### 23.01.31
- 직원 
  - 직원은 직원이 등록
  - 강사, 학생 등록/삭제
  - 직원 본인 정보(비밀번호) 수정

- 강사
  - 로그인
  - 본인 정보(비밀번호) 수정
  - 시험 문제(객관식) 등록
  - 시험 문제 수정

- 학생
  - 로그인
  - 본인 정보(비밀번호) 수정
  - 시험 응시
  - 답안지 입력 및 보기
  - 점수 확인
